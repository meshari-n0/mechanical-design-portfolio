# ⚙️ Joint System

The robotic arm movement is controlled using different mate constraints configured in the Onshape assembly.

| Joint Area | Mate Type | Description |
|-------------|-------------|-------------|
| Base Structure | Fastened | The base covers are rigidly fixed to the main base structure |
| Base Rotation | Revolute | Allows the circular platform to rotate the entire robot arm |
| Shoulder Joint | Revolute | Connects the base to the first arm segment |
| Elbow Joint | Revolute | Enables bending motion between the arm links |
| Intermediate Link | Cylindrical + Revolute | Allows both rotation and slight sliding motion between link components |
| Wrist Joint | Revolute | Controls orientation of the end-effector |
| Gripper Mechanism | Gear Mate | Synchronizes the rotation of the gripper gears |

# Shock Absorber – CAD Assembly (Onshape)

A fully modeled and assembled **shock absorber** designed in **Onshape**, built from technical drawings and assembled with proper mates to simulate realistic motion.

---

## Project Overview

This project presents a complete CAD assembly of a shock absorber, focusing on:

 **Mechanical design accuracy**
 **Clean assembly structure**
 **Correct mate setup & motion behavior**
 **Realistic compression/extension (suspension-like)**

---

## Components Included

The assembly contains the following parts:

- **Base**
- **Top Rod**
- **Spring**
- **Nut / Adjustment Gear**

Each part was modeled individually and then assembled using Onshape mates.

---

## Assembly & Motion

Mates used to achieve realistic behavior:

- **Cylindrical Mate** → rotational alignment  
- **Slider Mate** → vertical compression & extension  
- **Screw Mate** → threaded movement simulation  
- **Fastened + Planar Mates** → fixed constraints & stability  

This enables the shock absorber to behave similarly to a real suspension system.

---

## Design Goals

- Apply correct **mechanical constraints**
- Practice **multi-part CAD assembly**
- Simulate **linear spring motion**
- Improve skills in **Onshape Assembly & Mate Connectors**

---

## Tools Used

- **Onshape (Cloud-based CAD)**
- **Mate Connectors**
- **Assembly Mates**
- **Parametric Part Design**

# DOF-Robot-Arm-Forward-Kinematics-3D-Analysis
This project presents the mathematical modeling and numerical analysis of a 3-DOF robotic arm using Forward Kinematics.

![WhatsApp Image 2026-02-19 at 6 39 11 AM](https://github.com/user-attachments/assets/d275cc62-f7a9-40a7-9045-c23605b9fc11)
# Forward Kinematics – 3 DOF Robot Arm (3D)

## Link Lengths
L1 = 15 cm

L2 = 10 cm

L3 = 4 cm

## Assumed Joint Angles
θ1 = 45°

θ2 = 65°

θ3 = -70°

---

## Forward Kinematics Equations

x = 

L1*cos(θ1)

L2*cos(θ1 + θ2)

L3*cos(θ1 + θ2 + θ3)

y = 

L1*sin(θ1)

L2*sin(θ1 + θ2)

L3*sin(θ1 + θ2 + θ3)

z = L1 + L2*sin(θ2) + L3*sin(θ2 + θ3)

φ = θ1 + θ2 + θ3

---

## Numerical Calculation

# Link 1
x1 = 15*cos(45°) = 10.61

y1 = 15*sin(45°) = 10.61

# Link 2
x2 = 10*cos(45° + 65°)

x2 = 10*cos(110°) = -3.42

y2 = 10*sin(110°) = 9.40

# Link 3
x3 = 4*cos(45° + 65° - 70°)

x3 = 4*cos(40°) = 3.1

y3 = 4*sin(40°) = 2.6

---

## Final End Effector Position

x = 10.61 - 3.42 + 3.1 = 10.29 cm

y = 10.61 + 9.4 + 2.6 = 22.61 cm

z = 15 + 10*sin(65°) + 4*sin(65° - 70°)

z = 15 + 9.06 - 0.35

z ≈ 23.71 cm

φ = 45° + 65° - 70° = 40°

---

## Final Result

x ≈ 10.29 cm  

y ≈ 22.61 cm  

z ≈ 23.71 cm  

orientation φ = 40°

# ⚙️ Gear Transmission Design (1:18)

In addition to the robotic arm assembly, a **spur gear transmission system** was designed to achieve a **1:18 reduction ratio**.

The gears were generated using the **Custom Spur Gear Feature in Onshape**, allowing parametric creation of gears with different tooth counts.

---

## 🔧 Gear Configuration

The system consists of four spur gears:

| Gear | Teeth |
|-----|------|
| Gear 1 | 5 |
| Gear 2 | 10 |
| Gear 3 | 15 |
| Gear 4 | 90 |

---

## 🔢 Gear Ratio

The total gear reduction achieved in this system is:

1 : 18

This means the output gear rotates **18 times slower** than the input gear while increasing torque.

---

## ⚙️ Gear Assembly & Motion

After designing the gears, they were assembled in **Onshape Assembly Studio**.

The gears were connected using mechanical mates to simulate real motion.

### Assembly Constraints Used

- **Revolute Mate** → allows each gear to rotate around its axis  
- **Gear Mate** → synchronizes the rotation between the gears  

This setup allows the gears to rotate together while maintaining the defined gear ratio.

# 🧩 Assembly Structure

The planetary gear system consists of several interacting components:

1. Ring Gear  
2. Sun Gear  
3. Planet Gear 1  
4. Planet Gear 2  
5. Planet Gear 3  
6. Planet Carrier (Fixed Reference)

All gears were generated using the **Custom Spur Gear Feature** and assembled in **Onshape Assembly Studio**.

---

# ⚙️ Gear Motion System

The system uses mechanical mates to simulate realistic gear motion.

| Component | Mate Type | Description |
|-----------|-----------|-------------|
| Sun Gear | Revolute | Rotates around the central axis |
| Planet Gears | Revolute | Rotate around their individual shafts |
| Planet Carrier | Fixed | Holds the planet gears and acts as the reference frame |
| Gear Interaction | Gear Mate | Synchronizes motion between interacting gears |
| Ring Gear | Gear Interaction | Engages internally with the planet gears |
